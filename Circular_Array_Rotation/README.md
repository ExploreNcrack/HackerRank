# Circular Array Rotation
## Problem From HackerRank
**link**: https://www.hackerrank.com/challenges/circular-array-rotation/problem
<br /><br />John Watson performs an operation called a right circular rotation on an array of integers, [a<sub>0</sub>, a<sub>1</sub>, a<sub>2</sub> ....., a<sub>n-1</sub>]. After performing one right circular rotation operation, the array is transformed from [a<sub>0</sub>, a<sub>1</sub>, a<sub>2</sub> ....., a<sub>n-1</sub>] to [a<sub>n-1</sub>, a<sub>0</sub>, a<sub>1</sub> ....., a<sub>n-2</sub>].<br />Watson performs this operation **k** times.
